SP: Software for Semidefinite Programming

The SP software package is no longer available from this website. Interested users are advised to use one of the more recent semidefinite programming packages, for example,
  • DSDP
  • SeDuMi
  • SDPT3
  • SDPA
  • CSDP
  • YALMIP
See Christoph Helmberg's semidefinite programming website for a comprehensive and up-to-date list.

SP, released in 1994, was the first freely distributed SDP package. It used a Matlab/C implementation of Nesterov and Todd's primal-dual potential reduction method. The package is outdated by now, and superseded by the more efficient codes listed above.
